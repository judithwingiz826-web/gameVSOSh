# Problem Bank

Файл `problems.jsonl` содержит 530 стартовых задач с решениями и 4-ступенчатыми подсказками.

## Распределение
- arithmetic: 200
- algebra: 120
- combinatorics: 80
- geometry: 50
- number_theory: 80

Есть как неолимпиадные базовые задачи, так и задачи с `olympiad.track = municipal`.
