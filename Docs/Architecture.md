# Архитектура минимального каркаса

## Выбранный движок
**Unity HDRP** (версия редактора: 2022.3.20f1).

## Базовый игровой цикл
1. **Main Menu** — вход в игру.
2. **Student Profile** — создание/подтверждение профиля ученика.
3. **Hub Map** — выбор города и темы.
4. **Module Loading** — загрузка контента выбранного модуля.
5. **Theory Screen** — прохождение теории.
6. **Practice Screen** — решение задач.
7. **Boss Assessment** — итоговый мини-босс.
8. **Completed** — фиксация результатов.

## Сохранения
- Локальный профиль: `student_profile.json`.
- Переносимый профиль: JSON payload (экспорт/импорт).
- Фиксируются:
  - пройденные темы,
  - результаты по темам,
  - слабые места по скиллам.
