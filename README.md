# gameVSOSh — Unity HDRP Skeleton

Проект инициализирован как каркас под **Unity HDRP** в корне репозитория.

## Ключевая фича для подготовки к олимпиадам
- Задачи в `problems` (контенте) теперь привязаны к целевым олимпиадам/трекам и подтемам (`sourceTrack`, `subtopic`).
- В состоянии профиля хранится отдельная аналитика по каждому треку: точность, среднее число попыток и динамика изменений.
- В прогрессе доступен отчёт «готовность к треку» в процентах с расшифровкой слабых подтем.
- Добавлен режим мини-диагностики (8–12 задач) с итоговым отчётом и рекомендациями по усилению слабых мест.

## Каталоги
- `Game/` — bootstrap и управление игровым циклом.
- `Content/` — контент вертикального среза (1 город, 1 тема, теория, задачи, мини-босс).
- `Systems/` — системные подсистемы, включая сохранения.
- `UI/` — экраны (меню, профиль, хаб, теория, практика).
- `MathCore/` — генерация/проверка математических задач.
- `Progression/` — прогресс и слабые места.
- `Assessments/` — сессии задач и итоговые проверки.
- `BuildScripts/` — утилиты подготовки проекта.
- `Docs/` — документация по архитектуре и вертикальному срезу.

## Быстрый запуск
1. Открыть репозиторий через Unity Hub (версия из `ProjectSettings/ProjectVersion.txt`).
2. Дождаться импорта пакетов.
3. Подключить скрипты из каталогов как `Assets/Scripts` (или использовать symlink/UPM local package).
4. Запустить сцену bootstrap и проверить переходы: Menu → Profile → Hub → Theory → Practice.
