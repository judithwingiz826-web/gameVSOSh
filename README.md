# Math Odyssey: Olimpiad Trainer (Prototype)

Интерактивный тренажёр подготовки к олимпиадам по математике (СПб город, Курчатов, ВСОШ и перечневые) без Unity и без сборщиков.

## Что уже есть
- Полностью на `HTML + CSS + JavaScript`.
- Работает сразу: можно открыть `index.html` или запустить простой Python-сервер.
- Мобильная адаптация (телефоны/планшеты).
- Треки: СПб город, Курчатов, ВСОШ, перечневые.
- Теория + адаптивная практика + подсказки.
- Проверка ответов: числа, дроби, `да/yes/true`.
- Сохранение прогресса в `localStorage` + кнопка сброса.

## Запуск (без Unity)
### Вариант 1: просто открыть
Откройте файл `index.html` в браузере.

### Вариант 2: через Python
```bash
python3 -m http.server 4173
```
Открыть: <http://localhost:4173>

## Где хранится прогресс
Ключ `localStorage`: `math_odyssey_state_v2`.
Сброс: кнопка **«Сбросить прогресс»** в профиле.

## Ограничения MVP
- Это веб-прототип, не полноценный AAA-проект.
- Набор задач пока стартовый и требует расширения.
