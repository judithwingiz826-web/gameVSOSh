# Math Odyssey: Olimpiad Trainer (Prototype)

Интерактивный 3D-веб-прототип игры для подготовки к олимпиадам по математике (СПб город, Курчатов, ВСОШ и перечневые).

## Что уже есть
- 3D-сцена на `three.js` с анимированным фоном.
- Система профиля ученика 10 класса, старт «с нуля».
- Треки: СПб город, Курчатов, ВСОШ, перечневые.
- Теория по темам + адаптивная практика.
- Многоуровневые подсказки и мгновенная проверка.
- Счёт прогресса и рекомендации по слабым темам.

## Запуск
```bash
python3 -m http.server 4173
```
Открыть: <http://localhost:4173>

## Ограничения текущего MVP
Это не финальный AAA-продукт, а функциональный вертикальный срез:
- контент ограничен стартовым набором;
- 3D-графика веб-уровня;
- без серверной аналитики.
